# BabbleCast assets (source of truth)

Edit these files directly — no build script required.

| File | Use |
|------|-----|
| `bbcicon.png` | Master app icon (hand-cropped) |
| `icon.png` | 512×512 launcher icon for desktop + Android (`buildozer.spec`) |
| `splash.png` | Horizontal logo splash (desktop + Android presplash) |
| `logo-full.png` | Full banner reference |

When updating the icon: edit `bbcicon.png`, then copy/scale to `icon.png` (512×512, centered on black).
